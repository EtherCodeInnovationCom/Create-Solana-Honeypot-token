## 🚀 什么是陷阱代币（Honeypot Token）？

陷阱代币是一种用户可以**购买**但无法**卖出**的代币。通过智能合约中的特殊代码逻辑阻止卖出操作。通常用于安全测试、研究或其他特定场景。请谨慎使用。

---

## 🧰 本指南能帮您做什么？

借助我们的脚本和资源，您可以：
- 在 Solana 网络上创建新的 SPL 代币
- 添加防止卖出的自定义逻辑
- 在 Raydium 上上架该代币
- 分析用户行为
